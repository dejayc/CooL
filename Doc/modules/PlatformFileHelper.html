Functions

fileExists (fileName, coronaPathType)
Indicates whether the specified file exists within the specified Corona path type.

Parameters

  • fileName: The file name, optionally containing a file path, to check for in the specified Corona path type.
  • coronaPathType: The type of Corona path in which to search for the specified file name. Valid values are "system.DocumentsDirectory", "system.ResourceDirectory", and "system.TemporaryDirectory". Defaults to "system.ResourceDirectory".

Usage

  • fileExists( "data.xml" ) -- true if "data.xml" exists in the system.ResourceDirectory directory.
  • fileExists( "data/data.xml" ) -- true if "data.xml" exists in the "data" subdirectory of the system.ResourceDirectory directory.
  • fileExists( "data.xml", system.DocumentsDirectory ) -- true if "data.xml" exists in the system.DocumentsDirectory directory.

Return value:

True if the specified file name exists within the specified Corona path type; otherwise, false.

See also:

  • getFileSystemPath
  • system.pathForFile
getFileSystemPath (fileName, coronaPathType)
Returns an absolute path for the specified file if it exists within the specified Corona path type; otherwise, nil.

Parameters

  • fileName: The file name, optionally containing a file path, to check for in the specified Corona path type.
  • coronaPathType: The type of Corona path in which to search for the specified file name. Valid values are "system.DocumentsDirectory", "system.ResourceDirectory", and "system.TemporaryDirectory". Defaults to "system.ResourceDirectory".

Usage

  • getFileSystemPath( "data.xml" ) -- Absolute path to "data.xml" in the system.ResourceDirectory directory.
  • getFileSystemPath( "data/data.xml" ) -- Absolute path to "data.xml" in the "data" subdirectory of the system.ResourceDirectory directory.
  • getFileSystemPath( "data.xml", system.DocumentsDirectory ) -- Absolute path to "data.xml" in the system.DocumentsDirectory directory.

Return value:

The absolute path for the specified file if it exists within the specified Corona path type; otherwise, nil.
