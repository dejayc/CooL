Modules

BaseApp The base class for CooL applications.
BaseConfig The base class for classes that inspect and manipulate configurations.
ClassHelper Convenience functions for performing object-oriented class and object manipulation.
DataHelper Convenience functions for performing data manipulation.
DisplayHelper Convenience functions for inspecting and manipulating the display.
DisplayManager Manages the display as configured by Corona and CooL.
FileHelper Convenience functions for files, file systems, file names, and directories.
FrameworkConfig Convenience methods to inspect and manipulate the state of the CooL configuration.
FrameworkDisplay Convenience methods to inspect and manipulate the state of the display, as configured by CooL.
PlatformConfig Convenience methods to inspect and manipulate the state of the Corona configuration.
PlatformDisplay Convenience methods to inspect and manipulate the state of the display, as configured by Corona.
PlatformFileHelper Convenience functions for files, file systems, file names, and directories in Corona.
SpriteSheet Represents Corona SpriteSheet objects.
SpriteSheetData Represents Corona SpriteSheet data.

globals Global convenience functions and variables.
