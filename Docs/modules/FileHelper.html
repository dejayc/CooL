getPathComponents (path)
Returns the base path, file name, file base name, and file extension components of the specified path.

Parameters

  • path: The path to split into separate components.

Usage

  • getPathComponents( "/" ) -- { "/", nil, nil, nil }
  • getPathComponents( "/path/to/" ) -- { "/path/to/", nil, nil, nil }
  • getPathComponents( "/path/to/file" ) -- { "/path/to/", "file", "file", nil }
  • getPathComponents( "/path/to/file.txt" ) -- { "/path/to/", "file.txt", "file", "txt" }
  • getPathComponents( "/path/to/file.txt.bak" ) -- { "/path/to", "file.txt.bak", "file.txt", "bak" }
  • getPathComponents( "/path/to/.file" ) -- { "/path/to/", ".file", ".file", nil }
  • getPathComponents( "/path/to/.file.txt" ) -- { "/path/to/", ".file.txt", ".file", "txt" }
  • getPathComponents( "/path/to/../" ) -- { "/path/to/..", nil, nil, nil }
  • getPathComponents( "/path/to/./" ) -- { "/path/to/.", nil, nil, nil }
  • getPathComponents( "/path/to/.." ) -- { "/path/to/../", nil, nil, nil }
  • getPathComponents( "/path/to/." ) -- { "/path/to/./", nil, nil, nil }

Return values:

  1. The base path of the specified path.
  2. The file name of the specified path.
  3. The file base name of the specified path.
  4. The file extension of the specified path. components of the specified path.

See also:

getRequirePath (filePath)
Translates the specified path into a format compatible with the Lua "require" function.

Parameters

  • filePath: The path to be translated.

Usage

  • getRequirePath( "myModule.lua" ) -- "myModule"
  • getRequirePath( "Classes/myModule.lua" ) -- "Classes.myModule"

Return value:

The specified path translated into a format compatible with the Lua "require" function.

FileHelper class fields.

Fields

  • CLASS_SEPARATOR: Indicates the character that separates Lua subdirectory names that appear in the package name string parameter that is passed to the Lua "require" function. Statically assigned the value "."
  • FILE_EXTENSION_SEPARATOR: Indicates the character that separates base file names from file extensions. Statically assigned the value "."
  • FILE_EXTENSION_SPLIT_PATTERN: The Lua pattern that can be used to identify file extensions from file names.
  • LUA_FILE_EXTENSION: Indicates the common file extension for Lua source code files. Statically assigned the value ".lua"
  • PATH_CURRENT_DIRECTORY: Indicates that convention for specifying the current directory in a path. Statically assigned the value "."
  • PATH_PARENT_DIRECTORY: Indicates the convention for specifying the parent directory in a path. Statically assigned the value ".."
  • PATH_SEPARATOR: Indicates the character that separates subdirectories in paths. Assigned based on the environment during runtime; usually either "/" or "\"
  • PATH_FILE_NAME_SPLIT_PATTERN: The Lua pattern that can be used to identify path and file names from paths that contain file names.
  • PATH_TRIM_PATTERN: The Lua pattern that can be used to identify the leading path from in paths that contain file names.
